# SmartDine Desktop Edition Documentation

## Overview
A complete Python full-stack GUI restaurant management system for desktop use.

## Features
- Multi-role login system (Admin, Waiter, Chef, Cashier)
- Menu management with categories
- Table booking and management
- Order workflow (Waiter → Chef → Ready)
- Billing and payment processing
- Sales analytics and reporting
- Offline operation with SQLite database

## Installation
1. Install requirements: `pip install -r requirements.txt`
2. Run the application: `python app/main.py`

## User Roles
- **Admin**: Manage users, menu, prices, reports
- **Waiter**: Take orders, assign tables
- **Chef**: View and manage kitchen orders
- **Cashier**: Handle payments and billing
